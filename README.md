# DELTARUNE · Traducción al español americano

Traducción no oficial de *DELTARUNE* al español americano, distribuida como complemento del mod [*Deltranslate*](https://github.com/Lazy-Desman/DeltranslatePatch) de Neprim. Este repositorio reúne los archivos de idioma (`lang/`) que el mod utiliza para sustituir los textos del juego original por su versión traducida.

La traducción se rige por la variante del español estándar americano (código `es_mx` en la configuración del mod). Se procura un registro claro, coherente y respetuoso con el tono y los matices del texto original.

## Instalación

Existen dos vías, según las preferencias de cada jugador.

### Vía recomendada — Instalador automático

Es la forma más sencilla y la indicada para la mayoría de los jugadores. Solo requiere tener *DELTARUNE* instalado en el equipo.

1. Descargar `InstaladorLetraDelta.exe` desde la [última versión del instalador](https://github.com/BroxiCoros/LetraDeltaInstaller/releases/latest).
2. Ejecutarlo. El instalador detecta automáticamente la carpeta del juego, descarga los archivos necesarios y aplica todo el parche.
3. Listo. A partir de ese momento, cuando se publique una versión nueva el mod avisará dentro del juego y descargará los cambios sin necesidad de reinstalar nada.

El instalador también ofrece la opción de **versión con bordes (NXRUNE)** durante la instalación, por si se prefiere esa variante visual.

Junto con el pack de español, el instalador descarga también el [pack de inglés](https://github.com/BroxiCoros/LetraDelta-EN). Así, una vez aplicado el mod, el juego permite alternar entre ambos idiomas desde su menú interno y conserva la opción de jugar en inglés cuando se desee.

### Vía manual — Reemplazo directo de archivos

Pensada para quienes prefieran no pasar por el instalador. Requiere tener *DELTARUNE* instalado y conocer su carpeta (normalmente en `Steam/steamapps/common/DELTARUNE`).

1. Descargar `lang.zip` de la [última versión](../../releases/latest).
2. Extraer su contenido dentro de la carpeta del juego, dejando que se cree o actualice la carpeta `lang/`.
3. Asegurarse de tener el mod *Deltranslate* aplicado al `data.win` (lo aplica el instalador o se puede aplicar a mano desde [`DeltranslatePatch`](https://github.com/BroxiCoros/DeltranslatePatch)).
4. Iniciar el juego con normalidad.

> En cada versión también está disponible `lang.7z`, con el mismo contenido pero en formato 7-Zip y reorganizado para coexistir con otros idiomas. El instalador automático usa el `.7z`; los usuarios manuales suelen preferir el `.zip` por sencillez y compatibilidad.

## Reporte de errores y sugerencias

Los reportes se reciben preferentemente a través del servidor de Discord del proyecto, donde existen canales dedicados a errores de traducción, fallos gráficos y comentarios generales.

Quien no cuente con Discord, o prefiera hacerlo por GitHub, puede abrir un [*issue*](../../issues) describiendo el problema con el mayor detalle posible. Conviene indicar siempre la versión de la traducción y el capítulo afectado.

## Contribuciones

El equipo trabaja de forma coordinada a través del servidor de Discord. Quien desee participar como traductor, revisor o artista deberá presentarse allí. No se aceptan *pull requests* directos en este repositorio: los archivos se generan mediante un proceso interno que requiere contexto adicional.

## Repositorios del proyecto

*LetraDelta* se compone de cuatro repositorios coordinados:

| Repositorio | Contenido |
|------|-----------|
| **[LetraDelta](https://github.com/BroxiCoros/LetraDelta)** | Este repositorio. Pack de español (`lang/`). |
| **[LetraDelta-EN](https://github.com/BroxiCoros/LetraDelta-EN)** | Pack de inglés, para mantener disponible el idioma original junto con el español. |
| **[DeltranslatePatch](https://github.com/BroxiCoros/DeltranslatePatch)** | *Fork* del mod *Deltranslate* con un refactor propio. Provee el `scripts.7z` que parchea el juego. |
| **[InstaladorLetraDelta](https://github.com/BroxiCoros/InstaladorLetraDelta)** | Instalador `.exe` para Windows que coordina los tres anteriores. |
